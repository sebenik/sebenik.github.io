svg2base64

svg2base64 is a Node.js package that enables conversion of scalable vector graphics (svg) to base64 encoded images and outputs them directly to stdout. Package avoids saving any temporary files and instead uses streams to pipe data to destination. Among other benifits, this allows simple server side svg conversion from PHP, Python, etc.

  • npmjs: https://www.npmjs.com/package/svg2base64
  • GitHub: https://github.com/sebenik/svg2base64

CondorSoar

CondorSoar is a monitoring web application for HTCondor usage statistics.

It is inspired by The CondorView Client Contrib Module. CondorSoar adds new features and replaces Java applet with modern & interactive data visualizations using D3.js.

  • Demo: http://sebenik.github.io/CondorSoar/
  • GitHub: https://github.com/sebenik/CondorSoar
